# Security Policy

## Supported versions

| Version | Supported |
|---|---|
| 3.0.x | ✅ |
| < 3.0 | ❌ |

## Reporting a vulnerability

If you discover a security vulnerability, please email rkriad585@gmail.com
instead of opening a public issue. We will respond within 48 hours and
work with you to resolve the issue promptly.
